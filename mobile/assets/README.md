# Assets Placeholder

This directory contains placeholder assets for the claw-slack mobile app.

For a production build, you would need:
- icon.png (1024x1024) - App icon
- adaptive-icon.png (1024x1024) - Android adaptive icon  
- splash.png (1284x2778) - Splash screen image
- favicon.png (32x32) - Web favicon

The app will use default Expo assets for now.
